# t3Math

Tatty3D's cross-platform math module for stable ```mathematical```, ```matrix```, ```vector``` calculations. 



### ShortComing

- t3Vector3's ```rotate``` and ```map``` API is not avaliable.
  
- t3Vector4 and t3Matrix is not avaliable.
